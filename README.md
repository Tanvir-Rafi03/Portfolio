# Tanvir Rafi — Portfolio

A cyberpunk-themed personal portfolio built with React and Vite. Features smooth animations, interactive UI components, and a fully responsive design.

**Live Site:** [tanvirrafi.vercel.app](https://tanvirrafi.vercel.app)

---

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Animated intro with particle network background |
| **About** | Skills marquee with category filters, tech stack |
| **Projects** | Project showcase with live demo and GitHub links |
| **Services** | Services offered with interactive cards |
| **Contact** | Contact form (Formspree) and social links |

---

## Features

- Cyberpunk / HUD-inspired UI design
- Particle network animated background
- Custom cyberpunk cursor
- Boot screen with decrypt animation
- Command palette (Ctrl+K)
- Activity monitor widget
- Smooth scroll via Lenis
- Fully responsive — desktop, tablet, mobile
- Deployed on Vercel

---

## Tech Stack

- **React 19** — UI components
- **Vite 7** — Build tool and dev server
- **React Router** — Client-side routing
- **Lenis** — Smooth scrolling
- **Formspree** — Contact form backend
- **React Icons / Font Awesome** — Icon sets
- **CSS3** — Custom animations and responsive layout

---

## Getting Started

```bash
# Clone the repo
git clone https://github.com/Tanvir-Rafi03/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
